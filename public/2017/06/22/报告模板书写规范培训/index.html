变量

用{ { } }来修饰变量，内部的值会被渲染成文本。比如 我有{ { 1 + 1 } }个苹果会被渲染成 我有2个苹果。

通常我们会提前设置好一些变量，比如，我们在写模板的时候，为了方便起见，可以先设pro_code=44，即广东省的省代码，这些变量的设置可以在word的常量表里。那么我们在需要用到省代码的地方，可以写成{ { pro_code } }，如此便能在不同的省报告中使用不同的省代码了。
